  • Forum 1: Cloud Computing for Business
  • Forum 2: Cloudsourcing and Outsourcing
  • Forum 3: Cloud Computing for SMBs/SMEs
  • Forum 4: The Leading Cloud Vendor
  • Forum 5: Future Opportunities for Cloud Computing
  • Forum 6: The Next Generation Cloud Software and Services
  • Forum 7: Cloud DataBase and Info Management Systems
  • Forum 8: Social & Mobile Cloud Computing
  • Forum 9: Cloud Datacenter
  • Forum 10: Cloud Security and Cloud Monitoring
  • Forum 11: Cloud Computing and Virtualization
  • Forum 12: Cloud Storage
  • Forum 13: Cloud Computing and Big Data
  • Forum 14: Cloud Computing and Video Game
  • Forum 15: Cloud Terminal Device
  • Forum 16: Cloud Computing Adoption
  • Forum 17: Platform as a service (PaaS)
  • Forum 18: Infrastructure as a Service (IaaS)
  • Forum 19: Applications/Software as a Service (SaaS)
  • Forum 20: Cloud Deployment Models and Architectures

In addition to the scientific program, there will be a Matchmaking activity specially designed as well, which is to provide opportunities for the participants from China and the world to have face-to-face meetings on technical introduction, transfer, purchase, sales, and cooperation. If your research achievements are promising, both domestic and industrial parties relative to your projects will contact you to know if it is possible to cooperate with you for mutual advancement.
